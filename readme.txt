== Installation ==
La instalación y activación del plugin WP-CDP Connect es rápida y sencilla. A continuación, te indicamos los pasos que debes seguir para empezar a utilizarlo en tu sitio web:
1. Descarga el archivo .zip del plugin WP-CDP Connect desde nuestro sitio web o desde el repositorio oficial de WordPress.
2. Descomprime el archivo descargado en tu ordenador para obtener la carpeta "wp-cdp-connect".
3. Sube la carpeta "wp-cdp-connect" al directorio "/wp-content/plugins/" de tu sitio web utilizando un cliente FTP o mediante el gestor de archivos de tu hosting.Una vez que hayas subido la carpeta "wp-cdp-connect" correctamente, accede a tu sitio web y ve al menú "Plugins" en la interfaz de administración de WordPress.
4. Busca el plugin "WP-CDP Connect" en la lista de plugins instalados y haz clic en "Activar".
5. Una vez que el plugin esté activo, verás una nueva pestaña dentro de la interfaz de administración de WordPress. Allí podrás configurar y personalizar la integración con tu CDP y empezar a recopilar y gestionar los datos de tus clientes de manera eficiente.
